# Weather Widget

A Vue.js weather widget web component that displays current weather information for multiple cities.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Build the widget in development mode with watch:

```bash
npm run dev
```

## Build

Build the widget for production:

```bash
npm run build
```

## Testing

Run tests:

```bash
npm test
```

Run tests in watch mode:

```bash
npm run test:watch
```

## Integration

Add the widget to your HTML page:

```html
<weather-widget></weather-widget>
<script src="dist/weather-widget.js"></script>
```

## Requirements

- Node.js 18+
- npm 9+
